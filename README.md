# genpw

Простой генератор паролей на C с гибкой настройкой типов символов.

## Что это?

Консольная утилита для генерации случайных паролей. Можно выбрать длину и типы символов, которые будут использоваться.

## Как использовать?

```bash
./genpw длина [флаги]
```

**Флаги:**
- `-n` — включить цифры (0-9)
- `-c` — включить строчные буквы (a-z)
- `-C` — включить заглавные буквы (A-Z)
- `-s` — включить специальные символы (!@#$%^&* и т.д.)

**Примеры:**

```bash
# Пароль из 12 символов со всеми типами символов
./genpw 12

# Только цифры, 8 символов
./genpw 8 -n

# Буквы и цифры, 16 символов
./genpw 16 -n -c -C

# Только специальные символы, 10 символов
./genpw 10 -s
```

**Минимальная длина пароля:** 4 символа

## Как собрать?

### Требования
- GCC (компилятор C)
- Make

### Сборка

```bash
# Скомпилировать программу
make

# Запустить
./bin/release/genpw 12 -n -c -C -s
```

### Запуск тестов

```bash
make test
./bin/test/genpw_test
```

## Технологии

- **C** — основной язык
- **Make** — система сборки
- **ctest** — фреймворк для тестирования
- **GitHub Actions** — CI/CD

## Что реализовано?

- Генерация паролей заданной длины
- Выбор типов символов через флаги
- Гарантированное наличие хотя бы одного символа из каждого выбранного типа
- Алгоритм Фишера-Йетса для перемешивания символов
- Разделение на библиотеку (`libgenpw`) и исполняемый файл
- Юнит-тесты для проверки функциональности
- Автоматическая сборка и тестирование через CI
